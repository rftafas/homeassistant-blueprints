# Home Assistant Blueprints

This repository contains Home Assistant blueprints for automations.

## Blueprints

- n_way_irq.yaml:
  - N-Way Interrupts — Sets multiple switches to control a light entity.
    [![Add to my Home Assistant](https://my.home-assistant.io/badges/blueprint_import.svg)](https://my.home-assistant.io/redirect/blueprint_import/?blueprint_url=https://raw.githubusercontent.com/rftafas/homeassistant-blueprints/main/n_way_irq.yaml)
- climate-automation.yaml
  - Simple Climate Automation, with setpoint, presence and external reading.
    [![Add to my Home Assistant](https://my.home-assistant.io/badges/blueprint_import.svg)](https://my.home-assistant.io/redirect/blueprint_import/?blueprint_url=https://raw.githubusercontent.com/rftafas/homeassistant-blueprints/main/climate-automation.yaml)
- shutter_helper_automation.yaml
  - Automation for no-feedback RF controllers (using TasmotaRF)
    [![Add to my Home Assistant](https://my.home-assistant.io/badges/blueprint_import.svg)](https://my.home-assistant.io/redirect/blueprint_import/?blueprint_url=https://raw.githubusercontent.com/rftafas/homeassistant-blueprints/main/shutter_helper_automation.yaml)
- timed-light.yaml
  - Light Timer. Turns off after configurable time (day and night times)
    [![Add to my Home Assistant](https://my.home-assistant.io/badges/blueprint_import.svg)](https://my.home-assistant.io/redirect/blueprint_import/?blueprint_url=https://raw.githubusercontent.com/rftafas/homeassistant-blueprints/main/timed-light.yaml)


## Usage

Click the button next to each blueprint to import it directly into your Home Assistant instance.

See each YAML file for details and configuration options.
